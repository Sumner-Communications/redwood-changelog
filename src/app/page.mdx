## Platform Improvements & Bug Fixes {{ date: '2024-12-02T00:00Z' }}

We're excited to announce a comprehensive update to Wholesale Central with numerous improvements across the platform. This release focuses on enhancing the buyer and supplier experience, fixing critical bugs, and improving overall platform stability.

### <SparkleIcon /> Bug Fixes

- Fixed supplier location not displaying correctly in search results.
- Resolved issue where categories were not showing all available locations.
- Fixed Analytics not data from today.
- Corrected product creation timestamp not being stored properly.
- Fixed order failures caused by email delivery issues.
- Resolved payment validation errors on checkout.
- Fixed a bug preventing purchasing Featured Seller.
- Corrected membership pricing display accuracy on supplier dashboard.
- Fixed navigation on the website.
- Resolved a bug which caused cart to be empty temporarily.

### <SparkleIcon /> Improvements

- Added "How to reply" text guidance for email leads from buyers to suppliers.
- Enhanced image validation for validating products with invalid images.
- Contacting suppliers from product pages now includes product reference in the email.
- "Sell on Wholesale Central" header link now hidden for logged-in users.
- Other minor improvements and bug fixes.
